# Setup

Install [nodejs](https://nodejs.org/en/download/)
- [how to](https://phoenixnap.com/kb/install-node-js-npm-on-windows)

Execute in terminal:
> npm init -y

> npm i -D playwright

> npm install -D jest

> npm i -D @applitools/eyes-playwright
